Shuffle Stuff

The algorithm randomly selects pieces from each file you load and you can make it draw more or less from certain files by increasing the number (1-10) next to the file. Add as many files as you want but make sure they are in plain text format (.txt). The seperator character is what the program uses to divide up the text into pieces it can choose from. By default it is the new line/ return character (\n), but you can change it to something like a period if you're inputting paragraphs of text. If you reload the page, everything will reset.



Choose files and then click "Process"!
